Documentation

Learn how to use the Bot Control Panel

Quick Start Guide

1. Get Your Bot Token

Visit the Discord Developer Portal, create a new application, navigate to the "Bot" section, and copy your bot token.

2. Upload Your Bot Files

Go to the File Manager and upload your bot's files. You can:

  • Upload any file type (100MB max)
  • Upload ZIP files and extract them
  • Create new files and folders directly
  • Edit files with built-in editor

3. Configure Settings

Navigate to Settings and paste your bot token. The panel will securely store it and use it when starting your bot.

4. Start Your Bot

Return to the Console page and click "Start Bot". Your bot will install dependencies automatically and start running. Monitor the live console for any errors or logs.

5. Node.js Version

You can select different Node.js versions (18, 19, 20, 21) from the Node Version page. This is useful if your bot requires a specific Node.js version.

6. File Management (NEW!)

New in v2.0: Click the three-dot menu (â‹®) on any file to:

  • Edit: Edit text files directly in browser
  • Extract: Extract ZIP files automatically
  • Copy Path: Copy file path to clipboard
  • Delete: Remove files/folders

Use the "Create New" button to create files or folders instantly!
